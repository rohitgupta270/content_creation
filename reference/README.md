# Reference Materials

This directory contains style guides, visual references, and copywriting examples.

## Required Files

### Infographic Reference Images

Add 3 infographic reference images that define your visual style. These are used as `reference_image` when generating AI infographics via Kie.ai, ensuring all your content looks on-brand.

- `infographic-ref-1.jpeg` — Layout type 1 (e.g., radial/circular layout)
- `infographic-ref-2.jpeg` — Layout type 2 (e.g., scattered items around central headline)
- `infographic-ref-3.jpeg` — Layout type 3 (e.g., vertical flowing path)

**How to create these:**
1. Find 3 infographic styles you love (Matt Gray, Justin Welsh, Sahil Bloom, etc.)
2. Create or commission 3 infographics in YOUR brand colors and style
3. Each should have the SAME color palette but a DIFFERENT layout
4. Save them here as the reference images

### Carousel Reference Slides

Add example carousel slides to `carousel-ref/` to establish your carousel style:
- Save individual slides as `slide-00.jpg`, `slide-01.jpg`, etc.
- Add a `carousel-style-analysis.md` documenting the style patterns

### Personal Photo + Text Overlay References (Optional)

If you want posts with text overlaid on your photos:
- Add examples as `me-with-text-ref-1.jpeg`, etc.
