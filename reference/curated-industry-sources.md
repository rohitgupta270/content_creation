# Curated Industry Sources for Trend Intelligence

A curated list of 40+ industry sources for daily/weekly trend monitoring across AI/ML, Payments, Travel, and Product Management. Each source is selected for depth, strategic insight, and relevance to Rohit's content pillars.

**How to use this list:**
- Pick 5-10 sources per category and subscribe to their newsletters
- Set a daily/weekly review rhythm (15-20 minutes per day)
- When you find a signal, ask Claude: "Turn this into a viral replication post" or "Here's a trend, help me find the PM angle"
- Capture ideas in `/outputs/trend-signals-YYYY-MM-DD.md`

## Integration & Workflow

### How to Monitor These Sources

**Daily check (5-10 minutes):**
- The Batch (AI)
- FinTech Inside (Payments)
- PhocusWire (Travel)
- 1-2 PM sources of choice

**Weekly deep dive (20-30 minutes):**
- 3-4 sources from each category
- Look for patterns: what's being discussed across multiple sources?
- Capture emerging trends in `outputs/trend-signals-YYYY-MM-DD.md`

**Monthly review (1 hour):**
- Re-read high-signal pieces
- Identify which sources are most valuable (prune the noise)
- Brainstorm post ideas from trends/signals

### Tools to Aggregate

Use a news aggregator to centralize:
1. **Feedly** (RSS aggregator) — add all newsletter RSS feeds
2. **Substack** — subscribe directly to Substack newsletters
3. **Google News** — create sections for each domain + set alerts
4. **Twitter/X** — follow newsletter authors, industry analysts

### Template for Capturing Signals

When you find a signal, save it with this structure:

```
## [Signal Title]
**Source:** [Newsletter/Publication]
**Date:** [YYYY-MM-DD]
**Category:** [AI / Payments / Travel / PM]

**Signal:** [What's happening]
**Why it matters:** [Business/product implication]
**PM angle:** [How Rohit can approach this as content]
**Post type:** [Viral replication / Trend surfing / Pain point]
```
